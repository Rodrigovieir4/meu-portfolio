1. **Cores**: Edite as variáveis CSS em `css/style.css`
2. **Conteúdo**: Modifique o texto em `index.html`
3. **Traduções**: Atualize o arquivo `js/translations.js`
4. **Imagens**: Substitua arquivos na pasta `assets/`
